> Paste any YouTube link → get a clean AI-powered summary in seconds.

## Features
- 3 summary styles: Quick Bullets, Detailed, Study Notes
- Works with any YouTube video that has captions
- Powered by Google Gemini AI

## Tech Stack
- Python
- Streamlit
- youtube-transcript-api
- Google Gemini API

## Run Locally
pip install youtube-transcript-api google-generativeai streamlit python-dotenv
Add your GEMINI_API_KEY to a .env file
streamlit run app.py
